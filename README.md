
# Cafe65

인천지역 커피 프랜차이즈 브랜드 Cafe65 홈페이지 입니다.  

## 기능

- [x] 모바일 반응형 - media query 적용
- [x] 메인 - 이미지 슬라이더, 아코디언 메뉴, 동영상 삽입
- [x] 매장안내 - 위도, 경도를 활용한 구글 지도 API 연동 
- [x] 메뉴안내 - 메뉴별 상품 진열
- [x] 팝업창 
- [x] 가맹 전화번호 - 화면 좌측 하단 고정
